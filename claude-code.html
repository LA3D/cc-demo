Getting Started

Getting Started with Claude Code

Official Homepage: https://www.anthropic.com/claude-code

Installation:

First, install Node.js 18+ (macOS with Homebrew):

brew install node

Then install Claude Code:

npm install -g @anthropic-ai/claude-code

Requirements: - macOS, Linux, or Windows (via WSL) - Claude Pro ($17/month) or Max (starting $100/month) - Terminal access

The Terminal Advantage

Agent Workflow

  • Problem Analysis: Transforms requirements → specifications
  • Architecture Design: Creates system designs
  • Task Breakdown: Architecture → implementable tasks
